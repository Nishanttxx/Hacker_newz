Core features

Live Top Stories
Fetches fresh IDs from the HN API on every load. Each story's metadata loads concurrently for a snappy list experience.
Nested Comments
Recursively loads and renders the full comment tree. Indentation makes threading clear at a glance, with collapsible threads.
Detail Screen
Tap a story to see full metadata, the external URL, and all first-level comments parsed from the kids[] array.

Built with
Firebase
& Flutter.

The official Hacker News API runs on Firebase Realtime Database. All network calls — story lists, item metadata, and comment threads — handled natively in Dart using http and JSON parsing.

Flutter 3.x Dart Firebase REST Android iOS dart:convert http package
hn_service.dart
// Top stories endpoint
const baseUrl = 'hacker-news.firebaseio.com';

Future<List<int>> fetchTopIds() async {
  final res = await http.get(
    Uri.https(baseUrl, '/v0/topstories.json'));
  return List<int>.from(jsonDecode(res.body));
}

// Item detail + comments
Future<HNItem> fetchItem(int id) async {
  final res = await http.get(
    Uri.https(baseUrl, '/v0/item/$id.json'));
  return HNItem.fromJson(jsonDecode(res.body));
}
